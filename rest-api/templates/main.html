Exercise website

The website is up and running. You can query data from its REST API, where the following operations are supported. All GET requests will return the data in JSON format.

Get list of all users

Returns a list of all users with their usernames

GET /v1/users

Parameters

None

Responses

HTTP code 200 The list of all users with their usernames respectively. Sample response.
HTTP code 4xx, 5xx A description of the error

Get the access level of a user

Given a username returns that user's acces level on this site. Three authority values are possible: READ, WRITE, and ADMIN. All that applies will be returned.

GET /v1/accesslevels

Parameters

username Required. The username of the user to be queried as a String

Responses

HTTP code 200 A JSON object with the username and all the access rights that user has. (e.g. in case of admin access level it will return ["READ", "WRITE", "ADMIN"]. Sample response.
HTTP code 4xx, 5xx A description of the error
